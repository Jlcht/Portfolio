# JL's Portfolio

Live Demo: [https://frontend-project-c3a3e.web.app](https://frontend-project-c3a3e.web.app)

A modern, immersive portfolio website built with React and advanced 3D web technologies. This project showcases my professional journey, education, and interests through an interactive and visually engaging interface.

## 🛠 Technologies Used

- **Core**: React 19, Javascript
- **3D & Graphics**: Three.js, React Three Fiber, React Three Drei
- **Physics**: Cannon.js (@react-three/cannon)
- **Animations**: Framer Motion
- **Map/Globe**: React Globe GL
- **Styling**: CSS3, FontAwesome
- **Deployment**: Firebase Hosting

## ✨ Features

- **Interactive 3D Art Gallery**: An immersive First-Person Shooter (FPS) style gallery experience where users can walk around and view photography.
- **3D World Globe**: Interactive glove visualization showcasing travel locations.
- **Dynamic Animations**: Smooth page transitions and element animations using Framer Motion.
- **Responsive Design**: Fully optimized for various screen sizes.
- **Contact Integration**: Integrated contact form for easy communication.

## 🚀 Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/YourUsername/portfolio.git
    ```
2.  **Install dependencies**
    ```bash
    npm install
    ```
3.  **Run locally**
    ```bash
    npm start
    ```
